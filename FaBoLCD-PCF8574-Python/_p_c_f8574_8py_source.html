85 # When the display powers up, it is configured as follows:
86 #
87 # 1. Display clear
88 # 2. Function set:
89 # DL = 1; 8-bit interface data
90 # N = 0; 1-line display
91 # F = 0; 5x8 dot character font
92 # 3. Display on/off control:
93 # D = 0; Display off
94 # C = 0; Cursor off
95 # B = 0; Blinking off
96 # 4. Entry mode set:
97 # I/D = 1; Increment by 1
98 # S = 0; No shift
99 #
100 # Note, however, that resetting the Arduino doesn't reset the LCD, so we
101 # can't assume that its in that state when a sketch starts (and the
102 # LiquidCrystal constructor is called).
